# 小蛋饼专属导航

## 快速启动

双击 `启动导航站.bat`，自动启动服务器并打开浏览器访问 `http://localhost:3000/`

## 开机自启动设置

1. 右键 `启动导航站.bat` → 创建快捷方式
2. 按 `Win + R`，输入 `shell:startup`，回车打开启动文件夹
3. 把创建的快捷方式粘贴到启动文件夹即可

## 默认账户

- 用户名：`admin`
- 密码：`admin123`

## 目录说明

```
web/
├── index.html       首页
├── hotnews.html    今日热榜
├── login.html      登录/注册
├── admin.html      管理后台
├── about.html      关于本站
├── js/             JS脚本
├── poetry.json     诗词数据
└── 启动导航站.bat   启动脚本
```

## 注意事项

- 首次运行会自动安装依赖（需要网络）
- 服务器端口默认 `3000`
- 所有页面链接无需 `.html` 后缀
